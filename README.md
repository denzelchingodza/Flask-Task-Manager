# Flask-Task-Manager
A web-based task management application built with **Python and Flask**. This project allows users to efficiently manage tasks with features such as adding, editing, deleting, completing, and filtering tasks by status or priority.
## Features
- Add new tasks with **description**, **due date**, **priority**, and **estimated time**.
- Edit existing tasks directly from the dashboard.
- Delete tasks you no longer need.
- Mark tasks as completed.
- Filter tasks: **All**, **Pending**, or **Completed**.
- Priority badges with **color indicators** for quick visual reference.
- Responsive design using **Bootstrap 5**.
- Flash messages for user feedback after actions.

## Technologies Used

- **Python 3.10+**
- **Flask** – Web framework
- **Flask-SQLAlchemy** – ORM for SQLite database
- **Bootstrap 5** – Frontend styling
